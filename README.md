# DA5401_Assignment_8
DA5401 Data Analytics Lab : Assignment 8

The repo contains the code and the data used for the Assignment 8 of the course DA5401 : Data Analytics Lab. This assignment compares the ensemble techniques of Bagging, Boosting and Stacking to solve a complex, time-series-based regression problem.

The file hour.csv contains the hourly time series data of bike rental patterns that was used for the assignment. The dataset was downloaded from [UCI Machine Learning Repository - Bike Sharing Dataset](https://archive.ics.uci.edu/dataset/275/bike+sharing+dataset). The file A8_code_DA5401.ipynb contains the code solution to the questions of the assignment, along with the required observations and conclusion.
